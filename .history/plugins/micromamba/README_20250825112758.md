# micromamba plugin

This plugin is based on the conda plugin, and just changes the aliases to better fit the 'micromamba'
command.

The conda plugin provides [aliases](#aliases) for `micromamba`, usually installed via [anaconda](https://www.anaconda.com/) or [miniconda](https://docs.conda.io/en/latest/miniconda.html).

To use it, add `micromamba` to the plugins array in your zshrc file:

```zsh
plugins=(... micromamba)
```

## Aliases

| Alias    | Command                                 | Description                                                                     |
| :------- | :-------------------------------------- | :------------------------------------------------------------------------------ |
| `mna`    | `conda activate`                        | Activate the specified conda environment                                        |
| `mnab`   | `conda activate base`                   | Activate the base conda environment                                             |
| `mncf`   | `conda env create -f`                   | Create a new conda environment from a YAML file                                 |
| `mncn`   | `conda create -y -n`                    | Create a new conda environment with the given name                              |
| `mnconf` | `conda config`                          | View or modify conda configuration                                              |
| `mncp`   | `conda create -y -p`                    | Create a new conda environment with the given prefix                            |
| `mncr`   | `conda create -n`                       | Create new virtual environment with given name                                  |
| `mncss`  | `conda config --show-source`            | Show the locations of conda configuration sources                               |
| `mnde`   | `conda deactivate`                      | Deactivate the current conda environment                                        |
| `mnel`   | `conda env list`                        | List all available conda environments                                           |
| `mni`    | `conda install`                         | Install given package                                                           |
| `mniy`   | `conda install -y`                      | Install given package without confirmation                                      |
| `mnl`    | `conda list`                            | List installed packages in the current environment                              |
| `mnle`   | `conda list --export`                   | Export the list of installed packages in the current environment                |
| `mnles`  | `conda list --explicit > spec-file.txt` | Export the list of installed packages in the current environment to a spec file |
| `mnr`    | `conda remove`                          | Remove given package                                                            |
| `mnrn`   | `conda remove -y -all -n`               | Remove all packages in the specified environment                                |
| `mnrp`   | `conda remove -y -all -p`               | Remove all packages in the specified prefix                                     |
| `mnry`   | `conda remove -y`                       | Remove given package without confirmation                                       |
| `mnsr`   | `conda search`                          | Search conda repositories for package                                           |
| `mnu`    | `conda update`                          | Update conda package manager                                                    |
| `mnua`   | `conda update --all`                    | Update all installed packages                                                   |
| `mnuc`   | `conda update conda`                    | Update conda package manager                                                    |
